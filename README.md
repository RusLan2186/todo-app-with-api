## Instructions for Launch:
1. Clone the repository:

```bash
https://github.com/RusLan2186/todo-app-with-api.git
```


2. Install the necessary dependencies using:
   
```bash
npm install
```

3. Launch the project using:
   
```bash
npm start
```
